# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-12

### Added
- Initial release
- FCM provider with full support
- Single and bulk message sending
- In-memory queue with concurrency control
- Rate limiting with token bucket algorithm
- Retry engine with exponential backoff
- Analytics hooks for lifecycle events
- Metrics collection
- Message builder with fluent API
- TypeScript support with full type definitions
- Comprehensive error handling
- ESM and CJS dual build support

### Phase 2 (Planned)
- Expo provider implementation
- Message scheduling
- Persistent queue options
- Enhanced metrics and monitoring
